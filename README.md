# Tic-Tac-Toe Game

## Overview

This project is a Java-based implementation of the classic Tic-Tac-Toe game. It supports two-player gameplay and demonstrates the application of Object-Oriented Programming concepts and game logic implementation.

## Features

* Two Player Mode
* Dynamic Grid Support
* Winner Detection
* Input Validation
* Interactive Console Interface

## Technologies Used

* Java
* Object-Oriented Programming
* Control Structures
* Arrays

## Learning Outcomes

This project helped strengthen my understanding of:

* Game Logic Development
* User Input Validation
* Algorithm Design
* OOP Concepts
* Debugging Techniques
